# Касса — LLC PRO BIO FOODS

Веб-приложение для учёта кассовых операций компании. Работает локально в браузере, без бэкенда. Поддерживает несколько касс, две валюты (SUM/USD), категории, аналитику и синхронизацию через Google Sheets.

## Структура проекта

```
cash_app/
├── index.html    — разметка и подключение ресурсов
├── styles.css    — все стили (темы, сетки, модалки, адаптив)
├── script.js     — вся логика приложения
└── README.md     — этот файл
```

Внешние зависимости подключаются через CDN в `index.html`:
- **SheetJS (xlsx)** — экспорт в Excel
- **Google Fonts** — Inter, Sora

## Запуск

Просто открой `index.html` в браузере. Все три файла должны лежать в одной папке. Никакой сборки, npm или сервера не требуется — это статический сайт.

Если планируется использовать синхронизацию с Google Sheets, лучше открывать через локальный сервер (например `python3 -m http.server`), чтобы избежать ограничений CORS в некоторых браузерах.

## Возможности

- **📊 Панель** — текущие балансы по кассам, последние операции
- **➕ Операция** — добавление прихода/расхода с категорией и валютой
- **📋 История** — фильтрация по дате, категории, типу, поиск по заметкам
- **📈 Аналитика** — статистика по расходам, доходам и дням
- **💱 Конвертер** — USD ↔ SUM по заданному курсу, переводы между кассами
- **⚙️ Настройки** — темы (светлая/тёмная), начальные балансы, категории, импорт/экспорт JSON, настройка Google Sheets

## Хранение данных

Все данные хранятся в `localStorage` браузера. Это значит:
- Данные привязаны к конкретному браузеру и устройству
- Очистка кэша браузера = потеря данных
- **Регулярно делай экспорт в JSON** через раздел "Настройки" для бэкапа

Для синхронизации между устройствами используется интеграция с Google Sheets через Apps Script (поля `sheetId` и `deployUrl` в настройках).

## Разработка

Код разделён на три файла для удобства редактирования:
- Меняешь внешний вид → `styles.css`
- Добавляешь поле или вкладку → `index.html`
- Меняешь логику расчётов, валидацию, синхронизацию → `script.js`

CSS использует переменные в `:root` и `:root.dark-mode` — все цвета удобно править в одном месте в начале `styles.css`.
