## 💡 Основные функции

### 📊 Панель (Dashboard)
Главный экран с важной информацией:
- **Выбор касс** - переключайся между Наличными и Безналичными
- **Остаток денег** - текущий баланс в SUM и USD
- **Приходы/Расходы** - за сегодня
- **Последние операции** - быстрый просмотр
- **Кнопка обмена** - быстрый обмен валют

### ➕ Добавление операций
Простое добавление расходов и приходов:
- Выбор кассы (куда идёт операция)
- Тип: Расход или Приход
- Категория (автоматический выбор или вручную)
- Сумма с форматированием (`1 000 000`)
- Валюта (SUM или USD)
- Дата и время
- Описание/заметка
- **Шаблоны** - быстро добавляй повторяющиеся операции

### 📋 История операций
Полная история со множеством возможностей:
- **Поиск** по описанию
- **Фильтры** по категориям, типам, датам
- **Сортировка** по дате, сумме
- **Редактирование** - открой операцию и измени данные
- **Дублирование** - скопируй как основу новой операции
- **Удаление** - убери ошибочные операции
- **Экспорт в Excel** - скачай отчёт

### 📈 Аналитика
Подробный анализ расходов и приходов:
- По категориям (отдельно SUM и USD)
- По дням (последние 7 дней)
- Количество операций в каждой категории
- Клик на категорию - детальная информация

### 💱 Конвертер валют
Удобная конвертация:
- **Конвертер** - быстрая конвертация USD ↔ SUM
- **Установка курса** - вводи актуальный курс
- **Перевод между кассами** - движение денег между Наличными и Безналичными

### ⚙️ Настройки
Полный контроль над приложением:
- **Тема** - светлая (🌞) или темная (🌙)
- **Начальный остаток** - для каждой кассы отдельно
- **Google Apps Script** - синхронизация с облаком
- **Категории** - добавляй свои расходы и приходы
- **Резервная копия** - экспорт и импорт JSON

## 💵 Кассы

### По умолчанию 2 кассы:

1. **💵 Наличные** (cash)
   - Для наличных денег в офисе
   - Свой баланс SUM и USD
   - Свой начальный остаток

2. **🏦 Безналичные** (card)
   - Для банковских переводов
   - Независимый от наличных баланс
   - Отдельные операции

### Как пользоваться:

**На панели:** Нажимай кнопки для переключения:
- `📊 Все кассы` - видишь сумму обеих
- `💵 Наличные` - только эта касса
- `🏦 Безналичные` - только эта касса

**При добавлении операции:** Выбери кассу в самом начале

**В обмене валют:** Выбери в каком кассе менять

## 🔄 Синхронизация

### Зачем нужна?
Чтобы твои данные были в облаке (Google Sheets) и доступны с разных устройств.

### Как настроить?

1. **Создай Google Sheet:**
   - Открой [sheets.google.com](https://sheets.google.com)
   - Создай новую таблицу

2. **Добавь скрипт:**
   - Меню: Extensions → Apps Script
   - Вставь код (см. ниже)
   - Сохрани (Ctrl+S)

3. **Опубликуй скрипт:**
   - Нажми "Deploy" (вверху справа)
   - Выбери "New deployment"
   - Type: "Web app"
   - Execute as: твой аккаунт
   - Who has access: "Anyone"
   - Copy URL

4. **В приложении:**
   - Открой Настройки → Google Apps Script
   - Вставь ID Таблицы
   - Вставь Deployment URL
   - Нажми "Синхронизировать"

### Код для Apps Script:

```javascript
function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSheet();
    var data = JSON.parse(e.postData.contents);
    sheet.clear();
    sheet.getRange(1, 1, 1, 7).setValues([['ID', 'Дата', 'Тип', 'Категория', 'Сумма', 'Валюта', 'Описание']]);
    if (data.operations && data.operations.length > 0) {
      var values = [];
      for (var i = 0; i < data.operations.length; i++) {
        var op = data.operations[i];
        values.push([op.id, new Date(op.dateTime).toLocaleString('ru-RU'), op.type, op.category, op.amount, op.currency, op.description || '']);
      }
      sheet.getRange(2, 1, values.length, 7).setValues(values);
    }
    return ContentService.createTextOutput(JSON.stringify({success:true})).setMimeType(ContentService.MimeType.JSON);
  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({success:false, error: error.toString()})).setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSheet();
    var data = sheet.getDataRange().getValues();
    var operations = [];
    for (var i = 1; i < data.length; i++) {
      var row = data[i];
      if (row[0]) {
        operations.push({id:parseInt(row[0]), dateTime:new Date(row[1]).getTime(), type:row[2], category:row[3], amount:parseFloat(row[4]), currency:row[5], description:row[6]||''});
      }
    }
    return ContentService.createTextOutput(JSON.stringify({operations:operations})).setMimeType(ContentService.MimeType.JSON);
  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({operations:[], error:error.toString()})).setMimeType(ContentService.MimeType.JSON);
  }
}
```

## ❓ Часто задаваемые вопросы

### Где хранятся данные?
В `localStorage` браузера на твоем компьютере. При очистке истории браузера данные удалятся!

### Как сделать резервную копию?
1. Открой Настройки
2. Резервная копия → "Загрузить"
3. Сохранится файл `cash-app-data.json`

### Как восстановить данные?
1. Открой Настройки
2. Резервная копия → "Загрузить копию"
3. Выбери сохраненный JSON файл

### Можно ли удалить операцию?
Да! 
- В **История**: нажми на операцию → ✏️ Редактировать → Удалить
- На **Панели**: нажми на операцию → ✏️ Редактировать → Удалить

### Почему не синхронизируется?
Проверь:
1. Интернет подключен?
2. URL правильный? (должен содержать `/usercontent.googleapis.com/`)
3. Скрипт опубликован? (Deploy → New deployment)

### Как изменить курс валют?
- На Панели: кнопка "💱 Обменять валюту"
- В Конвертере: поле "Курс USD → SUM"
- При обмене: введи свой курс в модальном окне

### Можно ли добавить свои категории?
Да!
- Открой Настройки
- Категории расходов / приходов
- Введи название → нажми "+"

### Как удалить категорию?
- Настройки → Категории → нажми на название категории → крестик

## 🚨 Решение проблем

### Приложение не загружается
- Открой консоль (F12) - есть ошибки?
- Попробуй другой браузер
- Очисти кэш браузера (Ctrl+Shift+Delete)

### Данные не сохраняются
- Убедись что localStorage разрешен
- Попробуй инкогнито-режим
- Очисти кэш браузера

### Не синхронизируется с Google Sheets
- Проверь интернет соединение
- Убедись что Apps Script опубликован
- Проверь правильность URL

### Операция удалилась случайно
- Восстанови из резервной копии (Настройки → Загрузить копию)
